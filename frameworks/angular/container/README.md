# HERE Front-End Framework Starter: Angular Container

HERE Core Container with Angular 22+

Demonstrates basic context sharing and notification functionality, using current Angular best practices: standalone components, signals, [zoneless change detection](https://angular.dev/guide/zoneless) (no `zone.js`), built-in control flow (`@if`), and [`takeUntilDestroyed`](https://angular.dev/api/core/rxjs-interop/takeUntilDestroyed) for subscription cleanup.

For more comprehensive examples of HERE Core features, see the [workspace-starter](https://github.com/built-on-openfin/workspace-starter) repo.

---

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) and is kept up to date with Angular 22.

## Prerequisites

HERE Core RVM  
Node.js v22.12+

## Getting started

Install dependencies

```
npm install
```

Start the Angular dev server

```
npm start
```

Launch HERE Core (separate console)

```
npm run client
```
